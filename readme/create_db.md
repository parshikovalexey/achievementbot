Шаги по развертыванию базы данных
1. Создать БД на portal.azure.com (Базы данных SQL -> Добавить).
2. Получить строки подключения к созданной БД (Базы данных SQL -> <Созданная БД> -> Строки подключения).
3. Установить Cloud Explorer для Visual Studio.
4. Авторизоваться под аккаунтом Microsoft для получения доступа к Azure.
5. Подключиться к созданной БД, используя логин и пароль, прописанные при создании БД, добавить клиентский IP-адрес для получения доступа (Cloud Explorer -> Microsoft Imagine -> SQL Databases -> <Созданная БД> -> Открыть обозреватель объектов SQL Server).
6. Развернуть БД из модели Entity, запустив сгенерированный SQL запрос.
